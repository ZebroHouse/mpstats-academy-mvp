# Шорт-лист «Главный первый урок» под цель × маркетплейс

**Дата:** 2026-07-07
**Для:** методологи
**Зачем:** после онбординг-визарда ученику будем показывать **один главный урок** («▶ Начать с первого урока»), подобранный под его ответы в визарде (цель + маркетплейс). Сейчас 138 из 164 зарегистрированных проходят визард, но только 38 (23%) открывают хоть один урок — обрыв ровно на входе. Гипотеза: убрать развилку «диагностика / библиотека», дать один директивный урок-выигрыш.

**Что нужно от методологов:** по каждой ячейке матрицы (цель × маркетплейс) **подтвердить или заменить** предложенный «первый урок». Это черновик на основе реальных данных Академии — не финал.

---

## Как составлен черновик

- **Ось.** Цель визарда мапится в ось контента: `Реклама→MARKETING`, `Контент→CONTENT`, `Аналитика→ANALYTICS`, `Операции→OPERATIONS`, `Финансы→FINANCE`, `Продажи→MARKETING+ANALYTICS`, `Новый маркетплейс→beginner`.
- **Маркетплейс.** В системе Ozon-контент = только курс `05_ozon`; всё остальное считается WB. Поэтому для OZON выбор беден (см. флаги).
- **Ранжирование.** По реальной вовлечённости: `c` = завершений урока (досмотр ≥90%), `v` = ушедших в просмотр. Плюс приоритет коротким интро и уже размеченным `START`/`QUICK`.

⚠️ **Оговорка по данным:** метрики сняты с текущих зрителей (выборка небольшая — контент смотрят ~40 человек). Это сигнал, а не приговор. Редакторское чутьё «какой урок лучше как ПЕРВЫЙ» важнее цифр.

**Критерии хорошего первого урока:** (1) вводный/«зачем это», а не глубокая частность; (2) короткий — в идеале ≤15 мин, быстрый выигрыш; (3) бесплатный/доступный без оплаты; (4) сам по себе даёт ощущение пользы.

## Сквозные флаги (важно)

1. **OZON почти не покрыт.** Профильный Ozon-контент есть только по КОНТЕНТУ (`05_ozon`). Для рекламы/аналитики/операций/финансов Ozon-версий нет. Варианты: (а) временно реюзать WB-урок для Ozon-ветки, (б) поставить дозапись Ozon-интро в план. Совпадает с уже известным долгом по Ozon.
2. **Операции — слабая ось.** Нет очевидного «первого урока», топ — служебные боты. Нужен явный методологический выбор операционного интро.
3. **Ручная разметка тонкая.** Сейчас всего 5 уроков с бейджами `START`/`QUICK` (4 — аналитика, 1 — финансы). Чтобы витрина «Начни отсюда» и подбор первого урока работали шире — стоит разметить по 1 «герой-первый-урок» на каждую ячейку матрицы (7 целей × 2 МП).
4. **Длительность.** Часть сильных кандидатов 24–40 мин. Для первого касания предпочтительнее ≤15 мин; где основной кандидат длинный — в таблице дан короткий альтернативный.

## Что дальше (техническая часть)

После того как методологи проставят по одному «первому уроку» на ячейку — размечаем эти уроки (маппинг «цель×МП → lessonId», реюз механики бейджей) и подключаем: конец визарда → витрина `/dashboard` с героем «Твой первый урок» сверху (для холодного новичка) → сразу в этот бесплатный урок. Диагностика — кнопкой на витрине + мягкий призыв после досмотра первого урока.
