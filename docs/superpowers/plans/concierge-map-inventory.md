## Факты для авторинга (не перепутать)

- **Почта поддержки:** `clients@mpstats.academy` (с «s»). Форма `/support` пишет в CarrotQuest (событие `pa_support_request`), не шлёт письмо напрямую.
- **Отмена подписки:** `/profile` (блок «Подписка»), не `/billing`.
- **Сохранение контента:** только сердечко (Избранное). «Трек»/«добавить в трек» — устарело, не упоминать.
- **Цены:** из БД, не называть конкретных сумм в answer.
- **Реф-механика:** режим-зависимая (i1/i2) — формулировки без жёстких обещаний сроков там, где режим их меняет; дефолт i1 (друг 14 дн, пакет реферера +14).
- **Диагностика:** 15 вопросов, 5 осей, TARGET=70%, top-3 задачи, разбор ошибок по осям.
- **Квоты ассистента:** free 5/день, подписка/админ 50/день, burst 6/мин.
- **Все deepLink href проверяются** тестом `platform-map-hrefs.test.ts` (Task D3) — использовать только реальные роуты из таблиц выше.
