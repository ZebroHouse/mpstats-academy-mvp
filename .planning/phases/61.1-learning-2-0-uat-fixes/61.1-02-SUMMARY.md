# Phase 61.1 Plan 02: Персональный план — плоский redesign (задачи + уроки) Summary

Страница `/learn/plan` переведена с аккордеона диагностических секций на плоский вид «как Избранное»: блок «Рекомендованные задачи» (карточки-ссылки из `getRecommendedPath.addedJobs`, которые бэкенд отдавал, но страница не рендерила) + блок «Рекомендованные уроки» (плоский список, сорт по приоритету errors→deepening→growth→advanced, приоритет = бейдж над карточкой).

## What Changed

### Task 1 — Плоский список «Рекомендованные уроки» с бейджем приоритета (commit `8a3c5f3`)
- Удалён аккордеон секций: убраны `expandedSections`, `toggleSection`, `SECTION_DESCRIPTIONS`, `pluralLessons`, иконки/border/bg секций.
- `SECTION_STYLES` упрощён до `{ badgeColor }` (только цвета бейджей). Добавлен `SECTION_BADGES` (errors→«Ошибка», deepening→«Углубление», growth→«Рост», advanced→«Продвинутый»).
- Новый `flatLessons` мемо — флэттенит уроки всех диагностических секций в порядке приоритета, каждый урок несёт `sectionId` для бейджа.
- Рендер — единый `grid gap-3` LessonCard, над каждой карточкой span-бейдж с цветом из `SECTION_STYLES[sectionId].badgeColor`.
- Сохранено: `visibleTotal`/`visibleCompleted` + прогресс-бар (баг 6), `onRemoveFromTrack`, «Обновить план», re-diagnostic CTA, `data-tour="learn-sections"`. Проп `favorite` НЕ передаётся (план без сердечек).

### Task 2 — Блок «Рекомендованные задачи» из addedJobs (commit `2a793ba`)
- Новый `addedJobs` мемо из `recommendedPath.addedJobs`.
- Блок «Рекомендованные задачи» рендерится над «Рекомендованные уроки» только при непустом `addedJobs`. Каждая задача — `Link` на `/learn/job/{slug}` (Wrench icon, title, «N уроков · прогресс X/N»), без `FavoriteButton`.
- `planIsEmpty` расширен: план непуст при наличии диагностических уроков ИЛИ задач.
- `import { Wrench } from 'lucide-react'`.

## Deviations from Plan

### Verification deviation — typecheck не запущен в worktree
- **Issue:** План требует `cd apps/web && npx tsc --noEmit`. В этом git-worktree нет `node_modules` (зависимости не установлены; они есть только в основном репозитории MAAL). `tsc` падает на резолве `react`/`@playwright/test`/`@sentry/nextjs` и т.д. — это ошибки окружения, не моего кода.
- **Mitigation:** Все изменения type-consistent — доступ к данным через `any`, новых строго-типизированных API не вводилось; `flatLessons` типизирован как `{lesson, sectionId}[]`, бейдж-lookup'ы — `Record<string,string>`. Все grep-acceptance-критерии пройдены. Финальный typecheck — на CI / в основном дереве при merge (паттерн «деплой через основной репозиторий»).
