# Phase 61.1 Plan 01: Job manual-add → Favorite heart Summary

Replaced the manual «трек»-механику on the three live job surfaces (AgentSearch recommend «Решения», JobCard catalog, JobDetail page) with FavoriteButton(JOB). Manual adding of a job now always routes to «Избранное» (model A); the personal plan stays diagnostic-only. This also closes UAT bug 2а (shared isPending across 3 buttons) and 2.1 (button color) by removing the conflicting track button entirely.

## What was built

**Task 1 — JobCard + JobDetail (commit a211de1):**
- `JobCard.tsx`: removed the «В треке» badge. The «+ В трек» button already carried the `onAddToTrack && !job.isInTrack` guard, so catalog usage (JobCatalog passes no `onAddToTrack`) is heart-only. RecommendedJobsBlock (diagnostic) still passes `onAddToTrack`, so its add button is preserved.
- `job/[slug]/page.tsx`: dropped `addJobToTrack`/`removeJobFromTrack` mutations and both track buttons. Mounted `FavoriteButton itemType="JOB" itemId={job.id}` seeded via `favorite.isFavorited.useQuery`. All hooks hoisted above the `if (isLoading)` / `notFound()` early returns (Rules of Hooks, CLAUDE.md gotcha).

**Task 2 — AgentSearch recommend (commit 4c1dd19):**
- `AgentSearch.tsx`: removed `addJobMutation` plus the now-unused `getRecommendedPath` query and `trackedJobIds` memo. Each recommend card now mounts its own `FavoriteButton(JOB)`. State seeded via a single batch `favorite.isFavorited.useQuery` keyed on the recommend result's jobIds (`favoritedSet.has(\`JOB:${j.jobId}\`)`). Independent per-card hearts replace the single shared mutation → fixes bug 2а.

## Scope boundary honored

- `learning.addJobToTrack` / `removeJobFromTrack` router procedures NOT removed — still used by RecommendedJobsBlock and the rebuild flow.
- RecommendedJobsBlock NOT touched functionally — diagnostic jobs still go to the Plan via `addJobToTrack`.
- Word rename (трек→план etc.) deferred to plan 61.1-04 (W2) — not done here.
- `favorites/page.tsx` not modified — the JOB («Решения») section already renders favorited jobs.
