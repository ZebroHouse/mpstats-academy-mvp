# Phase 61.1 Plan 07: DAU/WAU/MAU Active-User Analytics (Frontend) Summary

Реализован фронтенд DAU/WAU/MAU аналитики на `/admin/analytics`: новый клиентский компонент `ActiveUsersSection` с карточками метрик, комбинированным line-чартом DAU/WAU/MAU и area-чартом липкости, плюс встраивание секции «Активные пользователи» в верх страницы.

## Tasks

### Task 1 — ActiveUsersSection component (commit `41af205`)
Создан `apps/web/src/components/admin/ActiveUsersSection.tsx` (client component):
- Вызывает `trpc.admin.getActiveUserStats.useQuery({ days })` с собственным period-селектором 7/14/30/90 (стиль кнопок переиспользован со страницы), дефолт 30 дней.
- **Stat cards** (StatCard): «Активные сегодня (DAU)», «За 7 дней (WAU)», «За 30 дней (MAU)» из `current.dau/wau/mau` с trend-подписью `+N/−N за период` (delta = current − previous, через `trend` проп StatCard, подпись скрывается при delta=0). Плюс «Липкость (DAU/MAU)» = `current.stickiness * 100` в процентах.
- **DAU/WAU/MAU чарт** — выбран ОДИН комбинированный recharts `LineChart` с тремя сериями + Legend: DAU `#2C4FF8` (mp-blue), WAU `#6CC40C` (mp-green), MAU `#F97316` (amber). ResponsiveContainer, dot=false, форматтеры дат как в ActivityChart.
- **Stickiness чарт** — `AreaChart` с `series[].stickiness` (ratio 0..1 → %), цвет `#EC4899` (mp-pink), тайтл «Липкость (DAU/MAU)», ось Y и tooltip форматируют `%`.
- Loading: skeleton (4 карточки + 2 чарта). Error: красная карточка с message.
- Empty/all-zero: каптион «Данные копятся с момента запуска — точные значения появятся в ближайшие дни».
- Honest caption под секцией про heartbeat + приблизительный бэкафилл.
- **Rules of Hooks**: все `useState`/`useQuery`/`useMemo` подняты выше любого early return (CLAUDE.md gotcha).

### Task 2 — Wire into page (commit `29080ea`)
- Импортирован и отрендерен `<ActiveUsersSection />` в новой секции с заголовком «Активные пользователи» в самом верху `page.tsx`, над существующими Summary stats / User Growth / Diagnostic чартами. Существующие чарты не тронуты.

## Chart approach chosen
Один комбинированный `LineChart` для DAU/WAU/MAU (3 цветные серии + легенда) — компактнее и нагляднее тренда, чем 3 отдельных ActivityChart. Липкость — отдельный area-чарт рядом в 2-колоночном grid.
