### Аналитика
| Вопрос 16  ·  Аналитика  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Вы хотите понять, в какие месяцы спрос на ваш товар растёт, а в какие падает. Какой показатель смотреть в первую очередь? |
| Варианты | A.  Динамику поисковых запросов по товару во времени B.  Текущую цену конкурентов C.  Размер своей рекламной ставки D.  Количество своих характеристик в карточке |
| Правильный | A — Динамику поисковых запросов по товару во времени |
| Почему так | Сезонность спроса прямее всего видна по динамике поисковых запросов: когда люди чаще ищут товар — спрос растёт. Цена и ставка влияют на продажи, но не показывают сезонную картину спроса. |


| Вопрос 17  ·  Аналитика  ·  Уровень 2 — Средний |  |
| Вопрос | Вы анализируете нишу на Ozon: продавцов становится больше, а суммарный объём продаж в категории не растёт. Как это правильно трактовать? |
| Варианты | A.  Ниша перенасыщается: доля каждого продавца падает, входить рискованно B.  Ниша растёт: больше продавцов — больше денег C.  Это нормальная ситуация, на неё не нужно реагировать D.  Объём продаж не связан с числом конкурентов |
| Правильный | A — Ниша перенасыщается: доля каждого продавца падает, входить рискованно |
| Почему так | Если игроков прибавляется, а общий спрос (объём продаж) стоит на месте, рынок делится на большее число участников — доля и выручка каждого падают. Это признак перенасыщения: вход требует сильного отличия, а не «как у всех». |


| Вопрос 18  ·  Аналитика  ·  Уровень 3 — Опытный |  |
| Вопрос | Вы выбираете между двумя нишами для входа. Ниша А: высокий спрос, но десятки сильных продавцов с отлаженными карточками. Ниша Б: спрос скромнее, но конкуренция слабая и карточки у всех посредственные. С чего начать оценку, а не гадать? |
| Варианты | A.  Сопоставить в каждой нише объём спроса с силой конкуренции и своей способностью выделиться — высокий спрос бесполезен, если не отвоевать долю B.  Всегда выбирать нишу с максимальным спросом C.  Всегда выбирать нишу с минимальной конкуренцией D.  Бросить монетку — обе одинаковы |
| Правильный | A — Сопоставить в каждой нише объём спроса с силой конкуренции и своей способностью выделиться — высокий спрос бесполезен, если не отвоевать долю |
| Почему так | Опытный подход — не смотреть на спрос и конкуренцию по отдельности, а оценивать их вместе с реалистичной оценкой, сможете ли вы выделиться. Большой спрос в нише с сильными игроками может не дать вам ни доли; умеренный спрос со слабыми карточками — это возможность зайти и обойти их контентом. Ответ — в сопоставлении, а не в одном факторе. |


### Маркетинг
| Вопрос 19  ·  Маркетинг  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Вы хотите показывать рекламу товара прямо в поисковой выдаче Ozon — там, где аудитория уже ищет покупку. Какой формат для этого предназначен? |
| Варианты | A. «Оплата за клик» с размещением в «Поиске» B. Медийные баннеры на главной странице C. Email-рассылка покупателям D. Рекламные посты во внешних соцсетях |
| Правильный | A — «Оплата за клик» с размещением в «Поиске» |
| Почему так | Чтобы показывать рекламу в выдаче, нужно выбрать: 1. Инструмент: «Оплата за клик». 2. Место размещения: «Поиск». Этот формат идеально подходит для поставленной задачи, так как: - Прямая выдача: товар показывается среди результатов поиска по ключевым запросам. - Горячая аудитория: объявление видят те, кто уже ищет такой товар и готов к покупке. - Оплата за результат: вы платите только за фактические переходы в карточку (CPC), а не за показы. |


| Вопрос 20  ·  Маркетинг  ·  Уровень 2 — Средний |  |
| Вопрос | У вас низкая выкупаемость товара. Как это влияет на эффективность рекламы и что с этим делать? |
| Варианты | A.  Реальная стоимость привлечения покупателя растёт — нужно учитывать выкуп в расчёте рекламы B.  Выкупаемость на рекламу не влияет C.  Низкая выкупаемость улучшает ROI рекламы D.  Достаточно просто поднять рекламный бюджет |
| Правильный | A — Реальная стоимость привлечения покупателя растёт — нужно учитывать выкуп в расчёте рекламы |
| Почему так | Если часть заказов не выкупается, вы платите за рекламу заказов, которые не превращаются в выручку: фактическая стоимость привлечения выкупленного клиента растёт, ROI падает. Рекламу надо считать на выкупленные заказы, а параллельно — работать над причинами невыкупа. |


| Вопрос 21  ·  Маркетинг  ·  Уровень 3 — Опытный |  |
| Вопрос | Реклама принесла 60 000 ₽ выручки при расходе 9 000 ₽. Чему равен ДРР и что это значит для решения? |
| Варианты | A.  15% — реклама в пределах нормы, можно масштабировать при сохранении рентабельности B.  15% — реклама убыточна, надо срочно отключать C.  6,7% — отличный результат D.  666% — ошибка в кампании |
| Правильный | A — 15% — реклама в пределах нормы, можно масштабировать при сохранении рентабельности |
| Почему так | ДРР = расходы ÷ выручка от рекламы × 100% = 9 000 ÷ 60 000 = 15%. Само по себе это приемлемый уровень для многих категорий; решение о масштабировании принимается с учётом маржи товара — если маржа покрывает 15% ДРР с запасом, кампанию можно усиливать. |


### Контент

| Варианты | A.  Расширенный блок описания с изображениями, таблицами и инфографикой B.  Видеозвонок с покупателем C.  Раздел отзывов покупателей D.  Рекламный баннер вне карточки |

| Вопрос 23  ·  Контент  ·  Уровень 2 — Средний |  |
| Вопрос | Как правильно использовать ключевые слова в описании товара на Ozon для SEO? |
| Варианты | A. Использовать ключевые слова с прямым вхождением, даже если это похоже на переспам — на Ozon это работает B. Перечислить максимум ключей подряд, не заботясь о читаемости C. Ключевые слова в описании не влияют ни на что D. Использовать только один ключ на всю карточку |
| Правильный | A — Использовать ключевые слова с прямым вхождением, даже если это похоже на переспам — на Ozon это работает |
| Почему так | На Ozon SEO отличается от Wildberries. Здесь алгоритм заточен на прямые вхождения ключевых фраз.  В аннотацию нужно вписать все эти фразы именно так, как они есть: • орехокол • орехокол для грецких орехов • орехокол универсальный • орехокол для фундука • орехокол для кедровых орехов • орехокол для фисташек Да, это выглядит как переспам. Да, текст становится нечитаемым. Но на Ozon это работает — карточка получает индексацию по всем ключам и выходит в топ. Читаемость описания для Ozon вторична. Покупатель принимает решение по фото, цене и отзывам, а не по красоте текста в аннотации. |


| Вопрос 24  ·  Контент  ·  Уровень 3 — Опытный |  |

| Варианты | A.  Сравнить CTR и конверсию карточки до и после изменений на сопоставимом периоде   B.  Поверить ощущению — раз кажется лучше, значит лучше C.  Спросить мнение знакомых, нравится ли им новая карточка D.  Посчитать количество новых фото — чем больше, тем лучше |
| Правильный | A — Сравнить CTR и конверсию карточки до и после изменений на сопоставимом периоде — ощущение «стало лучше» не доказательство |
| Почему так | Эффективность контента измеряется метриками, а не впечатлением. Опытный подход — сравнить CTR и конверсию до и после на сопоставимом трафике и периоде (в идеале A/B). «Кажется лучше» и мнение знакомых — не данные; решение принимается по цифрам карточки. |


### Операции
| Вопрос 25  ·  Операции  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | В чём ключевая разница между схемами FBO и FBS на маркетплейсе? |
| Варианты | A.  FBO — товар на складе маркетплейса; FBS — товар у вас, отправляете сами при заказе B.  Это одно и то же, разные названия C.  FBO — только для одежды, FBS — только для электроники D.  FBS быстрее FBO всегда и для всех товаров |
| Правильный | A — FBO — товар на складе маркетплейса; FBS — товар у вас, отправляете сами при заказе |
| Почему так | FBO (Fulfillment by Operator) — товар заранее на складе МП, доставку делает МП. FBS (Fulfillment by Seller) — товар хранится у продавца, он сам собирает и передаёт заказ при покупке. Разница — в том, у кого лежит товар и кто его отгружает. |


| Вопрос 26  ·  Операции  ·  Уровень 2 — Средний |  |
| Вопрос | Перед сезонным пиком продаж что нужно сделать с поставками и почему? |
| Варианты | A.  Заранее увеличить запас на складе — поставка и приёмка занимают время B.  Поставить товар в первый день сезона C.  Ничего не менять, маркетплейс сам управляет запасом D.  Уменьшить запас, чтобы снизить хранение |
| Правильный | A — Заранее увеличить запас на складе — поставка и приёмка занимают время |
| Почему так | Поставка на склад и её приёмка занимают дни, а спрос в сезон растёт резко. Если завозить товар в момент пика, продажи упрутся в нехватку остатка. Поэтому запас наращивают заранее, с учётом срока доставки и приёмки. |


| Вопрос 27  ·  Операции  ·  Уровень 3 — Опытный |  |
| Вопрос | У вас один ходовой товар. Поставщик в Китае, доставка долгая и иногда задерживается. Как снизить риск уйти в out-of-stock и потерять позиции? |
| Варианты | A.  Держать страховой запас и заказывать заранее с учётом срока и возможной задержки поставки, а не по факту обнуления остатка B.  Заказывать новую партию только когда товар почти закончился C.  Перевести товар на печать под заказ D.  Игнорировать риск — маркетплейс восстановит позиции сам |
| Правильный | A — Держать страховой запас и заказывать заранее с учётом срока и возможной задержки поставки, а не по факту обнуления остатка |
| Почему так | При долгой и нестабильной поставке нельзя ждать, пока остаток обнулится, — товар закончится до прихода новой партии, позиции просядут и будут восстанавливаться медленно. Решение — страховой запас плюс заказ заранее с буфером на задержку. Это операционное планирование, а не реакция по факту. |


### Финансы
| Вопрос 28  ·  Финансы  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Как считается маржинальность товара? |
| Варианты | A.  (Цена продажи − себестоимость) ÷ цена продажи × 100% B.  Цена продажи ÷ себестоимость C.  Себестоимость × количество продаж D.  Выручка − реклама |
| Правильный | A — (Цена продажи − себестоимость) ÷ цена продажи × 100% |
| Почему так | Маржинальность = (цена − себестоимость) ÷ цена × 100%. Она показывает, какую долю цены составляет валовая прибыль. Не путать с наценкой, которая считается от себестоимости, а не от цены. |


| Вопрос 29  ·  Финансы  ·  Уровень 2 — Средний |  |
| Вопрос | Вы начинающий селлер, расходы документально подтверждать сложно. Какой налоговый режим обычно проще для старта? |
| Варианты | A.  УСН «доходы» 6% — налог со всей выручки, без учёта расходов B.  УСН «доходы минус расходы» 15% всегда выгоднее C.  ОСНО с НДС D.  Режим выбирать не нужно |
| Правильный | A — УСН «доходы» 6% — налог со всей выручки, без учёта расходов |
| Почему так | УСН 6% «доходы» — простой старт: налог считается со всей выручки, расходы подтверждать не нужно. УСН 15% выгоден, только когда подтверждённые расходы превышают ~60% выручки; на старте при низких/неподтверждённых расходах 6% обычно проще и дешевле. [Налоговые правила проверять на актуальность.] |


| Вопрос 30  ·  Финансы  ·  Уровень 3 — Опытный |  |
| Вопрос | Закупка партии 200 000 ₽, ожидаемая выручка 350 000 ₽, переменные расходы (комиссия, логистика, реклама) 50 000 ₽. Чему равен ROI партии? |
| Варианты | A.  50% B.  75% C.  43% D.  100% |
| Правильный | A — 50% |
| Почему так | Прибыль = 350 000 − 200 000 − 50 000 = 100 000 ₽. ROI = прибыль ÷ инвестиции × 100% = 100 000 ÷ 200 000 = 50%. Частая ошибка — делить прибыль на выручку (получив «~29%») вместо деления на вложенные деньги. |
