### Аналитика
| Вопрос 1  ·  Аналитика  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | У вашего товара на Wildberries много показов в рекламе, но почти нет кликов. С чего правильнее всего начать разбор ситуации? |
| Варианты | A.  Посмотреть на главное фото и цену в выдаче — что видит покупатель до клика B.  Сразу поднять ставку CPM, чтобы получить больше показов C.  Добавить больше ключевых фраз в кампанию D.  Отключить кампанию и запустить новую |
| Правильный | A — Посмотреть на главное фото и цену в выдаче — что видит покупатель до клика |
| Почему так | Много показов при низком CTR — сигнал, что карточка не цепляет на этапе выдачи. Первое, что видит покупатель, — главное фото и цена; разбор начинается с них, а не со ставки (ставка влияет на охват, а не на привлекательность). |


| Вопрос 2  ·  Аналитика  ·  Уровень 2 — Средний |  |
| Вопрос | Вы хотите проверить актуальные рекламные ставки сразу по 50 поисковым запросам, чтобы спланировать кампанию. Какое действие в MPStats правильное? |
| Варианты | A.  Использовать инструмент «Проверка рекламных фраз» — он проверяет до 100 фраз за раз B.  Создать 50 отдельных кампаний и сравнить их ставки C.  Запросить ставки вручную по каждой фразе в кабинете WB D.  Посмотреть среднюю ставку по категории и применить ко всем |
| Правильный | A — Использовать инструмент «Проверка рекламных фраз» — он проверяет до 100 фраз за раз |
| Почему так | Инструмент «Проверка рекламных фраз» в MPStats позволяет проверить до 100 фраз одновременно и увидеть текущие ставки — это штатный способ для массовой проверки, в отличие от ручного перебора или создания лишних кампаний. |


| Вопрос 3  ·  Аналитика  ·  Уровень 3 — Опытный |  |
| Вопрос | Товар: за месяц 12 000 просмотров, конверсия в заказ 3%, выкуп 80%. Вы хотите выйти на 500 выкупленных единиц в месяц при той же конверсии и выкупе. Что показывает расчёт? |
| Варианты | A.  Текущих просмотров не хватает: нужно ~20 800 просмотров в месяц B.  Текущих просмотров достаточно, проблема только в цене C.  Нужно поднять конверсию до 10%, просмотры не важны D.  Невозможно посчитать без данных по марже |
| Правильный | A — Текущих просмотров не хватает: нужно ~20 800 просмотров в месяц |
| Почему так | 500 выкупов ÷ 0,80 = 625 заказов. 625 заказов ÷ 0,03 конверсии = 20 833 просмотра. Сейчас 12 000 — то есть просмотров почти вдвое меньше нужного. Это диагностический навык: связать цель по выкупам с воронкой через конверсию и выкуп. |


### Маркетинг
| Вопрос 4  ·  Маркетинг  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Вы впервые запускаете рекламу нового товара на Wildberries, у которого пока нет отзывов. Что разумнее сделать в первую очередь? |
| Варианты | A.  Сначала набрать первые отзывы, а уже потом активно вкладываться в рекламу B.  Сразу запустить рекламу на максимальном бюджете C.  Поставить минимальную цену и не запускать рекламу вообще D.  Дождаться, пока товар сам выйдет в топ органически |
| Правильный | A — Сначала набрать первые отзывы, а уже потом активно вкладываться в рекламу |
| Почему так | Реклама товара без отзывов работает слабо: покупателю не хватает социального доказательства, и трафик хуже конвертируется. Логичнее сначала получить первые отзывы, затем масштабировать рекламу — иначе бюджет тратится с низкой отдачей. |


| Вопрос 5  ·  Маркетинг  ·  Уровень 2 — Средний |  |
| Вопрос | Вы хотите равномерно собрать статистику по показам каждой фразы в новой кампании, прежде чем отключать слабые. Какую настройку автоминусации в MPStats выбрать? |
| Варианты | A.  Автоминусацию по параметру «Показы» B.  Автоминусацию по CTR с первого дня C.  Ручную минусацию без сбора статистики D.  Автоминусацию по частотности фразы |
| Правильный | A — Автоминусацию по параметру «Показы» |
| Почему так | Чтобы собрать примерно одинаковый объём показов по каждой фразе для честного сравнения, используется автоминусация по «Показам»: фразы минусуются после накопления заданного числа показов. Минусация по CTR с первого дня отрежет фразы до того, как наберётся статистика. |


| Вопрос 6  ·  Маркетинг  ·  Уровень 3 — Опытный |  |
| Вопрос | Две рекламные кампании на один товар.  Кампания А: высокий CTR, но мало заказов.  Кампания Б: CTR ниже, но заказов заметно больше. Бюджет ограничен, оставить нужно одну. Что выбрать и почему? |
| Варианты | A.  Кампанию Б: она приносит заказы, а конечная цель рекламы — продажи, а не клики B.  Кампанию А: высокий CTR всегда важнее C.  Любую — CTR и заказы не связаны D.  Отключить обе и запустить третью |
| Правильный | A — Кампанию Б: она приносит заказы, а конечная цель рекламы — продажи, а не клики |
| Почему так | Высокий CTR кампании А означает, что объявление кликают, но дальше покупка не происходит, — клики не превращаются в деньги.  Кампания Б приносит заказы, то есть бьёт в реальную цель. При ограниченном бюджете оставляют то, что даёт продажи, а не то, что даёт клики ради кликов. |


### Контент
| Вопрос 7  ·  Контент  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Какой элемент карточки товара на Wildberries сильнее всего влияет на то, кликнет ли покупатель по ней в выдаче? |
| Варианты | A.  Главное фото  B.  Полное описание товара C.  Раздел с характеристиками D.  Блок вопросов и ответов |
| Правильный | A — Главное фото |
| Почему так | В выдаче покупатель сначала видит главное фото и цену — именно фото определяет большую часть решения о клике. Описание и характеристики работают уже внутри карточки, после клика. |


| Вопрос 8  ·  Контент  ·  Уровень 2 — Средний |  |
| Вопрос | Вы используете нейросеть, чтобы улучшить инфографику уже работающей карточки. Что обязательно дать нейросети, чтобы результат был полезным? |
| Варианты | A.  Текущий визуал, метрики эффективности карточки и обратную связь покупателей B.  Только название товара C.  Только пример карточки конкурента D.  Достаточно общего запроса «сделай красиво» |
| Правильный | A — Текущий визуал, метрики эффективности карточки и обратную связь покупателей |
| Почему так | Нейросеть не знает контекста вашего товара. Для улучшения существующей карточки ей нужен полный контекст: текущий визуал, показатели (CTR, конверсия) и обратная связь покупателей — иначе она выдаст шаблон, не привязанный к вашей реальной проблеме. |


| Вопрос 9  ·  Контент  ·  Уровень 3 — Опытный |  |
| Вопрос | Конкурент с похожим товаром стабильно держится выше вас в выдаче. Цена и отзывы у вас сопоставимы. С чего начать, чтобы отыграть позицию через контент? |
| Варианты | A.  Разобрать карточку конкурента: его фото, инфографику, структуру описания и ключевые запросы — найти, что он показывает, а вы нет B.  Просто заказать новые красивые фото на свой вкус C.  Снизить цену намного ниже конкурента D.  Скопировать карточку конкурента один в один |
| Правильный | A — Разобрать карточку конкурента: его фото, инфографику, структуру описания и ключевые запросы — найти, что он показывает, а вы нет |
| Почему так | При сопоставимых цене и отзывах разница часто в контенте и SEO. Грамотный шаг — предметный разбор карточки лидера: какие выгоды он выносит на фото, как структурирует инфографику, по каким запросам ранжируется. Это даёт гипотезы под свой товар. Слепое копирование и «фото на вкус» без анализа — не диагностический подход. |


### Операции
| Вопрос 10  ·  Операции  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Вы работаете по схеме FBO на Wildberries. Что это означает для хранения и доставки вашего товара? |
| Варианты | A.  Товар лежит на складе маркетплейса, и маркетплейс сам его доставляет покупателю B.  Товар хранится у вас, и вы сами отправляете его при каждом заказе C.  Товар нигде не хранится, печатается под заказ D.  Доставку организует покупатель самостоятельно |
| Правильный | A — Товар лежит на складе маркетплейса, и маркетплейс сам его доставляет покупателю |
| Почему так | FBO (Fulfillment by Operator) — товар заранее поставляется на склад маркетплейса, и МП берёт на себя хранение и доставку. Это и отличает FBO от FBS, где товар лежит у продавца и он отправляет его сам при заказе. |


| Вопрос 11  ·  Операции  ·  Уровень 2 — Средний |  |
| Вопрос | Ваш ходовой товар закончился на складе WB (out-of-stock). Что произойдёт и как правильно реагировать? |
| Варианты | A.  Позиции в выдаче просядут; восстановление займёт время, поэтому важно не допускать обнуления остатка. Чем меньше по времени отсутствует на складе, тем быстрее его позиции восстановятся после поставки на склад.  B.  Ничего не изменится, позиции сохранятся до новой поставки C.  Товар автоматически переключится на FBS без вашего участия D.  Маркетплейс компенсирует упущенную выручку |
| Правильный | A — Позиции в выдаче просядут; восстановление займёт время, поэтому важно не допускать обнуления остатка |
| Почему так | При обнулении остатка товар теряет позиции, а после новой поставки ранжирование восстанавливается не мгновенно — нужно заново набирать статистику продаж. Поэтому ключевая операционная задача — планировать поставки так, чтобы не уходить в out-of-stock. |



| Вопрос 12  ·  Операции  ·  Уровень 3 — Опытный |  |
| Вопрос | Товар продаётся в среднем 8 шт/день, новая поставка едет 14 дней, страховой запас вы держите 30 шт. На каком остатке нужно делать перезаказ? |
| Варианты | A.  На 142 шт B.  На 112 шт C.  На 30 шт D.  На 8 шт |
| Правильный | A — На 142 шт |
| Почему так | Точка перезаказа = средние продажи в день × срок поставки + страховой запас = 8 × 14 + 30 = 142 шт. Если ждать остатка ниже, товар успеет закончиться до прихода поставки и уйдёт в out-of-stock. |


### Финансы
| Вопрос 13  ·  Финансы  ·  Уровень 1 — Новичок (базовый) |  |
| Вопрос | Какой из перечисленных расходов селлера на маркетплейсе относится к переменным (зависит от объёма продаж)? |
| Варианты | A.  Комиссия маркетплейса с каждой продажи B.  Аренда офиса C.  Зарплата постоянного бухгалтера D.  Стоимость закупленного оборудования |
| Правильный | A — Комиссия маркетплейса с каждой продажи |
| Почему так | Переменные расходы растут вместе с числом продаж. Комиссия МП и логистика начисляются за каждую проданную единицу — это переменные. Аренда, оклад, оборудование не зависят от объёма продаж — это постоянные расходы. |


| Вопрос 14  ·  Финансы  ·  Уровень 2 — Средний |  |
| Вопрос | Товар продаётся по 1 200 ₽. Себестоимость 400 ₽, комиссия WB 10% (120 ₽), логистика 80 ₽, реклама 50 ₽. Какая чистая прибыль с единицы? |
| Варианты | A.  550 ₽ B.  800 ₽ C.  650 ₽ D.  120 ₽ |
| Правильный | A — 550 ₽ |
| Почему так | Чистая прибыль = цена − все расходы = 1 200 − 400 − 120 − 80 − 50 = 550 ₽. Частая ошибка — вычесть только себестоимость (получив «800»), забыв про комиссию, логистику и рекламу. Комиссия указана как пример [проверить актуальную ставку по категории]. |




| Вопрос 15  ·  Финансы  ·  Уровень 3 — Опытный |  |
| Вопрос | Постоянные расходы 50 000 ₽/мес. Маржинальная прибыль с единицы товара 300 ₽. Сколько единиц нужно продать в месяц, чтобы выйти в ноль? |
| Варианты | A.  167 единиц B.  100 единиц C.  300 единиц D.  500 единиц |
| Правильный | A — 167 единиц |
| Почему так | Точка безубыточности = постоянные расходы ÷ маржинальная прибыль с единицы = 50 000 ÷ 300 = 166,7 ≈ 167 единиц. Меньше — убыток, больше — прибыль. Это базовый расчёт для решения «стоит ли держать товар». |
