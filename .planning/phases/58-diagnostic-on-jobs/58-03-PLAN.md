JobScore algorithm per D-02:
```
weakAxes = top-2 axes sorted by ascending correctRate (always 2 items, no threshold)
for each published job:
  intersection = job.axes ∩ weakAxes
  score = Σ over axis in intersection: (1 − userAxisCorrectRate[axis])
sort by score DESC, tiebreaker job.id ASC
take top-3
```

// applied as where: { marketplace: { in: allowedMarketplaces } }
